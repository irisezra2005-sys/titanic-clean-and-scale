# titanic-clean-and-scale 
This repository contains a complete, end-to-end data preprocessing and feature scaling pipline for the Titanic Dataset. The goal of this project was to transform raw, messy passenger data into a clean, normalized, and fully optimized format.

## Objective
Perform exploratory data analysis and preprocessing on the Titanic dataset.

## Dataset
Titanic Dataset downloaded from Kaggle

## Steps Performed
1. Loaded dataset using Pandas
2. Checked missing values and data types
3. Imputed missing values
4. Applied One-Hot Encoding
5. Scaled numerical features using StandardScaler
6. Saved processed Dataset

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab
